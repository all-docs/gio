GSimpleIOStream creates a GIOStream from an arbitrary GInputStream and GOutputStream. This allows any pair of input and output streams to be used with GIOStream methods.

This is useful when you obtained a GInputStream and a GOutputStream by other means, for instance creating them with platform specific methods as g_unix_input_stream_new() (from gio-unix-2.0.pc / GioUnix-2.0), and you want to take advantage of the methods provided by GIOStream.
