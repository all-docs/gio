Description [src]

Like g_dbus_proxy_call() but also takes a GUnixFDList object.

This method is only available on UNIX.

Available since: 2.30

This method completes asynchronously. Use g_dbus_proxy_call_with_unix_fd_list_finish() inside the GAsyncReadyCallback to obtain the result of the operation.
