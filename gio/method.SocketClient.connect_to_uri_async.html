Description [src]

This is the asynchronous version of g_socket_client_connect_to_uri().

When the operation is finished callback will be called. You can then call g_socket_client_connect_to_uri_finish() to get the result of the operation.

Available since: 2.26

This method completes asynchronously. Use g_socket_client_connect_to_uri_finish() inside the GAsyncReadyCallback to obtain the result of the operation.
