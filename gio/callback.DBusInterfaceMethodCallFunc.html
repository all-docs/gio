Description [src]

The type of the method_call function in GDBusInterfaceVTable.

interface_name may be NULL if not specified by the sender, although itâ€™s encouraged for the sender to set it. If unset, and the object has only one method (across all interfaces) matching method_name, that method is invoked. Otherwise, behaviour is implementation defined. See the D-Bus specification. It is recommended to return G_DBUS_ERROR_UNKNOWN_METHOD.
