Async version of g_app_info_launch_uris().

The callback is invoked immediately after the application launch, but it waits for activation in case of D-Busâ€“activated applications and also provides extended error information for sandboxed applications, see notes for g_app_info_launch_default_for_uri_async().

Available since: 2.60

This method completes asynchronously. Use g_app_info_launch_uris_finish() inside the GAsyncReadyCallback to obtain the result of the operation.
