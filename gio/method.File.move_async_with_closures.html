Description [src]

Version of g_file_move_async() using closures instead of callbacks for easier binding in other languages.

Available since: 2.82

This method completes asynchronously. Use g_file_move_finish() inside the GAsyncReadyCallback to obtain the result of the operation.

This method is renamed to g_file_move_async() in language bindings.
