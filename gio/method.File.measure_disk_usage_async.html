Description [src]

Recursively measures the disk usage of file.

This is the asynchronous version of g_file_measure_disk_usage(). See there for more information.

Available since: 2.38

This method completes asynchronously. Use g_file_measure_disk_usage_finish() inside the GAsyncReadyCallback to obtain the result of the operation.

This method is not directly available to language bindings.
